# Manim for pedestrian dynamics

Collection of some scripts to explain models of [pedestrian dynamics](https://pedestriandynamics.org/).

## Run 

```
manim -pql CFM.py
```

or to produce high quality video

```
manim -pqh CFM.py 
```
