# Collision Free Speed Model
Explanatory video of the [collision_free_speed_model](https://pedestriandynamics.org/models/collision_free_speed_model/) using manim.

## Run 

```
manim -pql CFM.py
```

or to produce high quality video

```
manim -pqq CFM.py 
```
